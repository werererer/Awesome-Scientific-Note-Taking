# Awesome-Scientific-Note-Taking
Planned: _iteratively_ collect papers and resources on the science of note-taking. Since nobody publishes under the header “note-taking,” the topic must be approached through its abstract representations across various fields.

The goal is to create a non-compulsory but helpful reading list for anyone who searches deeper theory about note taking. To not only blindly type out notes, but make it principled. Making it not only a craft but an art [^1].
